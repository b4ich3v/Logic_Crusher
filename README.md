# **Logic crusher**

| Binary logical operators | Supported alternatives |
|----------|----------|
| Disjunction | OR, or, ∨, |, || |
| Conjunction |  AND, and, &, ∧ |
| Exclusive or |  XOR, xor, ^, ⊕ |
| Equivalence | EQV, eqv, <=>, ↔ |
| Implication | IMP, imp, =>, → |
| Not AND | NAND, nand, !&, ¬& |
| Not OR | NOR, nor, !v, ¬∨ |

| Unary logical operators | Supported alternatives |
|----------|----------|
| Negation | NOT, not, !, ~, ¬ |
